# PCS LaTeX Demo Project

This folder is an Overleaf-ready demo manuscript for the PCS machine-learning project.

## Files

- `main.tex`: main LaTeX manuscript.
- `tables.tex`: consolidated table bodies used by `main.tex`.
- `references.bib`: minimal bibliography.
- `figures/pdf/`: PDF figures used directly by `main.tex`.
- `figures/png/`: PNG figure copies for non-LaTeX use.
- `figures/tiff/`: TIFF figure copies for submission or publication systems.

## Compile on Overleaf

1. Upload the entire `latex_demo_project` folder to Overleaf.
2. Set `main.tex` as the main document.
3. Compile with pdfLaTeX.

The text is intentionally concise and academic. It is a demonstration document for presenting the current pipeline and results, not a complete journal submission.
